# 🛒 Superstore Sales Analysis

An end-to-end Data Science project covering Exploratory Data 
Analysis, Machine Learning, and interactive Power BI dashboards 
on the popular Superstore dataset.

---

## 📂 Repository Structure

| Folder | Contents |
|--------|----------|
| `data/` | Raw & cleaned CSV files |
| `notebooks/` | EDA & ML Jupyter notebooks |
| `dashboard/` | Power BI dashboard screenshots |

---

## 📊 Notebook 1: EDA (`notebooks/01_EDA.ipynb`)
- Sales & Profit analysis by Region, Category, Segment
- Monthly Sales trend over 4 years
- Discount vs Profit impact analysis
- Identified loss-making sub-categories (Tables, Bookcases)

## 🤖 Notebook 2: ML Model (`notebooks/02_ML_Model.ipynb`)
- **Sales Forecasting** using Facebook Prophet (6-month forecast)
- **Profit Prediction** using Random Forest Classifier
- Feature Importance: Discount is the #1 profit killer
- Exported forecast & feature importance data for Power BI

## 📈 Power BI Dashboard
- Sales Overview: Revenue, Orders, Profit Margin KPIs
- Regional & Segment breakdown
- Geo map of sales by US state
- Forecast chart with upper/lower bounds

---

## 🔑 3 Key Business Insights
1. **Discount above 40% always leads to losses**
2. **Technology is the most profitable category**
3. **Q4 (Nov–Dec) shows peak sales every year**
