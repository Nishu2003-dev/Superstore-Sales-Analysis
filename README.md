# 🛒 Superstore Sales Analysis

## 📌 Project Overview
End-to-end data science project on the Superstore dataset — covering Exploratory Data Analysis, Machine Learning, and interactive Power BI dashboards.

## 📂 Project Structure
- `01_EDA.ipynb` — Exploratory Data Analysis
- `02_ML_Model.ipynb` — Machine Learning Model (Profit Prediction)
- `dashboard/` — Power BI screenshots

## 🔑 Key Insights
- Discount above 40% always leads to losses
- Technology is the most profitable category
- Q4 (Nov–Dec) shows peak sales every year

## 🛠️ Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn, Scikit-learn)
- Power BI
- Jupyter Notebook
